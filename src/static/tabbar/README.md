# TabBar 图标说明

此目录需要放置以下图标文件（建议尺寸：81px × 81px）：

- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标
- `me.png` - 我的未选中图标  
- `me-active.png` - 我的选中图标

**临时方案：**
如果暂时没有图标，可以：
1. 使用在线图标生成工具创建简单图标
2. 或者暂时使用纯文字tabBar（需要修改pages.json，移除iconPath配置）

**推荐图标风格：**
- 未选中：灰色 (#999999)
- 选中：主题色 (#667eea)
- 简洁的线条图标风格
